Durante o semestre do meu curso de Engenharia de Software, consegui adquirir diversas habilidades, dentre elas, a como desenvolver um banco de dados na minha matéria de Banco de Dados. Com isso, juntamente com as habilidades adquiridas na minha de Engenharia de Software, consegui desenvolver juntamente com meus grupo de colegas da faculdade um banco de dados completo, chamado de sabores do cerrado. Basicamente o objetivo deste projeto é conectar e facilitar a comercialização de produtos artesanais do cerrado brasileiro feitos pelos produtores e seus consumidores, onde o seu diferencial é que a sua entrega é feita apenas no DF e entorno. Esse marketplace é uma ideia em estágio inicial, mas que pode ser melhorado em muitos pontos ao decorrer do curso.
